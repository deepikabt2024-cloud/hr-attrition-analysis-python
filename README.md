# hr-attrition-analysis-python

This project analyzes employee attrition to identify key factors influencing turnover. Using Python (Pandas, Matplotlib, and Seaborn), the analysis explores demographic, financial, and work-related patterns that contribute to employees leaving the company.
The dataset contains 1470 employees, of which 237 employees resigned, resulting in an overall attrition rate of 16.1%.
